Bu proje, frontend'de React ve backend'de Node.js kullanılarak geliştirilmiştir. Geliştirme aşamasında sunucunun otomatik yeniden başlatılması için `nodemon` kullanılmaktadır.

## Projenin Kurulumu

### Gereksinimler
- Node.js (v20.11.1)
- npm (10.2.4)
- `nodemon`

### Adımlar

1. Cliet için:
   ```bash
   npm install
   npm start

2. Server için:
   ```bash
   npm install
   nodemon
